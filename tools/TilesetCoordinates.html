🎮 RPG Maker MZ Tileset Viewer

Load your project folder to view all tilesets and their coordinates

📋 How to Use

1. Click "Select RPG Maker MZ Project Folder" and choose your game's root folder
2. The tool will automatically load Tilesets.json and all tileset images
3. Select a tileset from the dropdown to view its tiles
4. Hover over any tile to see its coordinate (e.g., "A1 5" or "B10")
5. Click any tile to copy its coordinate to clipboard
6. Use these coordinates in your tileset notes:
    Single tile: <FeatureName: A1 5> or <FeatureName: B10>
    Multi-tile: <Castle: [B10, B11],[B20, B21]>

Tip for large projects: If you get an error, try selecting just the data folder instead of the root folder. Some browsers have file limits. Open the browser console (F12) to see detailed debug information about what files were found.
