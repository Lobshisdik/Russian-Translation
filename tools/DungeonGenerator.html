RPG Dungeon Generator (MZ - Layer 0 - B1 Stacked Walls)

Settings

ID from Database > Tilesets
Number of wall tiles stacked vertically in wall sections above floor. Requires a stackable wall tile.

Fixed Tile IDs (Layer 0 - Using B1 tiles 390, 391, 392):

  • Floor: 390 (Random B1 tile)
  • Wall Stack: 391 (Random B1 tile)
  • Ceiling Top: 392 (Random B1 tile)
Note: Assumes your B1 tileset has suitable graphics for these IDs for vertical stacking.
